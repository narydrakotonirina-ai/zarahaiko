# Zarahaiko — Landing page
zarahaiko - Plateforme de formation numérique et réseau de femmes leaders communautaires à Madagascar
## Fichier
`zarahaiko-landing.html` — page unique, autonome (HTML + CSS + JS inline, images encodées en base64). Aucune dépendance externe à part les polices Google Fonts (Fraunces, Work Sans, JetBrains Mono).

## Utiliser le fichier
- **Voir en local** : double-cliquer sur le fichier, il s'ouvre dans le navigateur.
- **Mettre en ligne** : héberger tel quel (Netlify, GitHub Pages, OVH, etc.) — un seul fichier à uploader.
- **Éditer le contenu** : ouvrir le fichier dans un éditeur de texte, tout le texte est directement dans le HTML (pas de CMS).

## Structure de la page
1. **Hero** — positionnement et accroche
2. **Le constat** — chiffres sur la fracture numérique à Madagascar (sourcés : Kepios, UIT, MWC 2025, DECIM)
3. **Mission & objectifs**
4. **Notre approche** — capital humain existant (AC, femmes leaders) + plateforme Open edX + kit terrain
5. **Le programme** — parcours SBC en 5 blocs, basé sur `BLOC0` à `BLOC4_MOOC_SBC.docx`
6. **Kit d'autonomisation numérique** — tablettes, connexion satellite, solaire, ordinateurs
7. **Résultats attendus**
8. **Structuration de l'association** — cadre légal (Ordonnance n°60-133 du 3 octobre 1960) et gouvernance
9. **Partenaires recherchés**
10. **Contact / footer**

## SEO
Meta description et balises Open Graph (titre, description) incluses dans le `<head>` du fichier — utilisées par les moteurs de recherche et par l'aperçu du lien sur les réseaux sociaux / messageries.

## Points à vérifier avant publication
- **Email de contact** : `contact@zarahaiko.mg` est un placeholder — à remplacer par l'adresse réelle (2 occurrences : section CTA finale + footer).
- **Chiffre "N" projets communautaires produits** (section Résultats attendus) : laissé volontairement en placeholder, à chiffrer.
- **Section légale** : résume l'Ordonnance n°60-133 (1960, modifiée en 1975) à titre informatif — à faire valider par un juriste avant toute démarche officielle de déclaration.
- **Logo** : version actuelle avec ombre portée grise, lisible sur fond clair (nav, footer) — un logo sur fond transparent sans ombre donnerait un meilleur rendu sur les fonds foncés.
